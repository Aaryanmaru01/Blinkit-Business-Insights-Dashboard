# Blinkit-Business-Insights-Dashboard
This dashboard provides insights into Blinkit's grocery sales performance, including total revenue, number of orders, item categories, and outlet performance. It helps analyze customer purchasing behavior and identify high-performing products and locations.

# 🛒 Blinkit Sales & Performance Dashboard
## 📌 Project Overview

#### This project is a Power BI dashboard built using Blinkit grocery data. The dashboard provides insights into sales performance, customer behavior, and product trends.

## 🎯 Objectives
#### Analyze total sales and number of orders
#### Identify top-performing products and categories
#### Compare outlet performance
#### Understand customer purchasing patterns

## 📊 Key Metrics
#### Total Sales
#### Number of Orders
#### Average Sales Value
#### Item Categories
#### Outlet Performance

## 📈 Dashboard Features
#### Sales by Outlet Type
#### Sales by Item Category
#### Location-based analysis
#### KPI cards for quick insights
#### Interactive filters and slicers
